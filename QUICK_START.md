# 标签生成功能 - 快速开始指南

## 5分钟快速入门

### 第1步：准备模板
1. 打开应用，进入"标签模板"列表
2. 点击"添加模板"按钮创建或编辑一个模板
3. 设计好你的标签模板后，保存模板

### 第2步：生成标签

#### 2.1 打开对话框
1. 在模板列表中找到你的模板
2. 点击"生成标签"按钮
3. 数据输入对话框会弹出

#### 2.2 填写共同信息
在第一部分中填写以下信息：

| 字段 | 示例 | 必填 |
|-----|------|-----|
| 产品名 | 产品A | ✓ |
| 材料编号 | MAT001 | ✓ |
| 材料描述 | 高强度塑料 | |
| 客户名称 | 客户1 | |
| 客户采购订单 | PO2026001 | |
| 交货时间 | 2026-05-26 | ✓ |
| 单位装箱数 | 10 | ✓ |
| 位置 | 仓库A-架1 | |

#### 2.3 添加批次信息
1. 点击"添加批次"按钮
2. 填写批号和数量
   - 批号：B001
   - 数量：25
3. 系统会自动计算：
   - 箱数：3（25÷10=2余5，所以3箱）
   - 起始箱号：1
   - 结束箱号：3
4. 可以继续添加更多批次

**批次信息示例：**

```
批次1：批号B001，数量20
  → 生成2条标签（各10个）

批次2：批号B002，数量25
  → 生成3条标签（10、10、5个）

总共生成：5条标签
```

#### 2.4 配置生成选项
在第三部分配置：
- **将多个批次汇总显示**：
  - 不选：每条标签显示自己的批号（B001或B002）
  - 选中：所有标签显示"B001 - B002"
  
- **交货时间显示格式**：
  - YYYY-MM-DD（2026-05-26）
  - DD/MM/YYYY（26/05/2026）
  - MM/DD/YYYY（05/26/2026）
  - YYYYMMDD（20260526）

### 第3步：打印或导出

#### 3.1 点击"打印表签"
系统会：
1. 验证所有数据
2. 生成标签数据
3. 弹出操作选择对话框

#### 3.2 选择操作

**打印标签**
- 调用浏览器打印功能
- 选择打印机进行打印
- 支持打印到PDF

**导出为 CSV**
- 下载为Excel兼容格式
- 方便后续处理或分享

**导出为 JSON**
- 下载为JSON格式
- 便于程序集成

**导出为 PDF**
- 下载为PDF文件
- 保持完整的视觉格式

## 常见场景

### 场景1：生成固定数量的标签

**需求**：生产30件产品A，每箱10件，需要3张标签

**操作步骤**：
1. 填写共同信息
   - 产品名：产品A
   - 单位装箱数：10
2. 添加1条批次
   - 批号：B001
   - 数量：30
3. 点击"打印表签"
4. 系统生成3条标签
5. 打印或导出

### 场景2：多批次混合生产

**需求**：
- 批次B001：20件产品A
- 批次B002：15件产品A
- 每箱10件

**操作步骤**：
1. 填写共同信息
   - 产品名：产品A
   - 单位装箱数：10
2. 添加批次信息
   - 批次1：B001，数量20 → 生成2条标签
   - 批次2：B002，数量15 → 生成2条标签（10+5）
3. 勾选"将多个批次汇总显示"
   - 所有标签批号显示为"B001 - B002"
4. 点击"打印表签"
5. 系统生成4条标签，全部打印或导出

### 场景3：有余数的装箱

**需求**：生产25件产品，每箱10件

**说明**：
- 25 ÷ 10 = 2 余 5
- 需要生成3条标签
- 前2条：每条10件
- 第3条：5件

**生成的标签数据**：
```json
[
  { "productName": "产品A", "unitsPerBox": 10, "boxNumber": 1 },
  { "productName": "产品A", "unitsPerBox": 10, "boxNumber": 2 },
  { "productName": "产品A", "unitsPerBox": 5,  "boxNumber": 3 }
]
```

## 技巧和最佳实践

### ✓ 使用建议

1. **保存常用模板**
   - 为不同的产品创建标准模板
   - 节省重复设计时间

2. **批量添加批次**
   - 一次性添加多个批次信息
   - 使用批号合并功能简化显示

3. **选择合适的日期格式**
   - 了解客户的日期格式偏好
   - 避免日期理解错误

4. **验证单位装箱数**
   - 确保填写正确的单位装箱数
   - 这直接影响箱数和标签数量的计算

5. **打印前预览**
   - 使用打印对话框的预览功能
   - 检查标签是否符合预期

### ✗ 常见错误

1. **单位装箱数填写错误**
   - ❌ 不填或填0
   - ✓ 填写实际每箱数量

2. **数量计算错误**
   - ❌ 25 ÷ 10 = 2（忽略余数）
   - ✓ 25 ÷ 10 = 2 余 5（需要3张标签）

3. **日期格式选错**
   - ❌ 不检查日期格式，导致显示不正确
   - ✓ 提前确认客户要求的日期格式

4. **忘记添加批次信息**
   - ❌ 直接点击"打印表签"
   - ✓ 至少添加一条批次信息

## 数据格式说明

### 生成的标签数据结构

```typescript
{
  productName: string;        // 产品名
  materialNumber: string;     // 材料编号
  materialDescription: string;// 材料描述
  customerName: string;       // 客户名称
  customerPO: string;         // 客户采购订单
  deliveryDate: string;       // 交货时间（已格式化）
  unitsPerBox: number;        // 本条标签的单位装箱数
  location: string;           // 位置
  batchNumber: string;        // 批号（如启用合并则显示范围）
  boxNumber?: number;         // 箱号
}
```

```csv
产品A,MAT001,高强度塑料,客户1,PO001,2026-05-26,10,仓库1,B001,1
产品A,MAT001,高强度塑料,客户1,PO001,2026-05-26,10,仓库1,B001,2

```

## 故障排除

### Q: 为什么生成的标签数不对？
**A:** 检查以下几点：
- 单位装箱数是否正确？
- 数量计算是否考虑了余数？
  - 数量÷单位装箱数，如果有余数需要+1

### Q: 为什么日期显示格式不对？
**A:** 
- 检查选择的日期格式是否正确
- 确保输入的日期格式有效（YYYY-MM-DD）

### Q: 导出的文件无法打开？
**A:**
- CSV：使用Excel或其他电子表格应用
- JSON：使用文本编辑器或JSON查看工具
- PDF：使用PDF阅读器

### Q: 打印时页面空白？
**A:**
- 检查浏览器的打印设置
- 尝试调整页面边距
- 使用"背景图形"选项
